Ai 국가 index 가져오기
-> Ai 국가 index 별 도시 가져오기 (하나씩 or 전부)
-> 건설, 군대 배치 이벤트 2개 함수로 생성 (두 함수 모두 파라미터는 도시)
	-> 건설은 건설 속도를 고려
	-> 군대가 패배할 경우 다음 군대는 더 크게 생성
-> 일정 턴 마다 두 이벤트 중 하나를 선택해서 실행
	-> 게임이 끝날 때 까지 반복
	-> 랜덤 인카운트 생성 예시 -> Random으로 0,1,2중 숫자 하나 생성
			               0일 때 건설, 1일 때 군대, 2일 때 스킵

-> AI의 군대 이동은 고유의 스크립트가 따로 존재해야함
	-> 군대 또한 국가 index가 존재해야함
